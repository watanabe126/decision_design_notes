# decision_design_notes
判断と設計のメモ

小さなシステムや運用について、
制約条件と判断内容だけを簡単に残すためのメモです。

評価・正解・最適解を導くことは目的としていません。
詳細な手順やノウハウの共有も目的ではありません。

その時点で何を観測し、
どのような前提で判断したかを
後から振り返れる形で残すことを目的としています。
