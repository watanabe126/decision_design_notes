# ゲーム進捗観測メモ（非診断）

このフォルダは、ゲームのセーブデータや進捗を
「性格診断」「適職診断」のために用いるものではありません。

ゲーム中の行動・中断点・選択傾向を **観測ログとして整理**し、
対話や振り返りの材料にするためのフレームワークです。

- 評価しない
- 断定しない
- ラベリングしない
- 事実と質問だけを残す

## 目的
- 「何が起きたか」を後から再現できる形で残す
- 次の行動（遊び方／声かけ／導線設計）に繋げる

## スコープ
- 本人の同意なしに個人の実データを公開しない
- 子どもの具体的ログはこのリポジトリに置かない（フレームのみ）

## ファイル
- save_data_signals.md：観測できる指標の一覧
- questions.md：評価を避ける質問テンプレ
