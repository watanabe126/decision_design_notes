## 搬送方式選択フロー（現場目線）

以下は「性能が高いか」ではなく、
**構造的に成立するか／成立しないか**を切るためのフローである。


```
---

Q1. ワーク同士の接触は許容できるか？
 ├─ YES → Q2
 └─ NO  → Q4

---

Q2. 工程は停止前提で成立するか？
 ├─ YES → インデックス装置 / フリーフロー / MDR
 └─ NO  → Q3

---

Q3. 工程時間の揺れはバッファで吸収できるか？
 ├─ YES → 3本コンベア + 待避線 / ゾーン制御MDR
 └─ NO  → Q4

---

Q4. ワークに意味のある位置精度が必要か？
 ├─ NO  → ベルト搬送 / MDR
 └─ YES → Q5

---

Q5. 加工・検査は定位置・定時間で完結するか？
 ├─ YES → 直線インデックス / 直線アクチュエータ
 └─ NO  → Q6

---

Q6. 外部軸との厳密同期（速度・位相）が常時必要か？
 ├─ YES → 非推奨（既存方式で再設計を検討）
 └─ NO  → Q7

---

Q7. 工程分岐・スキップ・個別ID管理が必須か？
 ├─ YES → リニアトラック（条件付き）
 └─ NO  → 直線インデックス / コンベアに戻る

---

Q8. 姿勢制御（多面加工・多方向検査）が必要か？
 ├─ YES → 非接触・多自由度搬送（極北事例）
 └─ NO  → リニアトラックは過剰

---
```
【注意】
- Q6 で YES の場合、リニアトラックは原則棄却
- Q8 まで進むケースは極めて稀
- 多くの設備は Q2〜Q3 で解が出る

このフローの目的は
「最適解を探す」ことではなく
**早く・安全に候補を捨てること**である。
